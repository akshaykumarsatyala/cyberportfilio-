## ⚡ Deploy (Free — 30 seconds)

### Option 1: Netlify Drop
1. Go to [netlify.com/drop](https://netlify.com/drop)
2. Drag the entire `akshay-cyber-portfolio` folder
3. Done — live URL instantly!

### Option 2: GitHub Pages
1. Upload folder to a GitHub repo
2. Go to Settings → Pages → Deploy from main branch
3. Your site is live at `username.github.io/repo-name`

### Option 3: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` inside the folder
3. Follow the prompts

---

## 🎨 Customize

- **Your name / links**: Edit `index.html` — search for `akshaykumarsatyala`
- **Theme colors**: Edit `css/style.css` — change `:root` variables
- **Add projects**: Copy a `.proj-card` block in `index.html` Room 3
- **Matrix speed**: Edit `setInterval(draw, 45)` in `js/matrix.js`
